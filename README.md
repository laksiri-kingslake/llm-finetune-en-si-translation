# LLM Fine-tune Guide

## Virtual Environment and Dependencies

1. Create
```bash
python3 -m venv .venv
```
2. Activate
```bash
source .venv/bin/activate
```
3. Deactivate
```bash
deactivate  
```
4. Install dependencies
```bash
pip install unsloth trl peft accelerate bitsandbytes torch
```
